# Loan Prediction Using Machine Learning

## Project Overview

This project predicts whether a loan application will be approved or rejected using Machine Learning classification algorithms. The project includes data preprocessing, handling missing values, encoding categorical variables, balancing the dataset using SMOTE, training multiple models, and evaluating their performance.

## Dataset

* Dataset: Loan Prediction Dataset
* Target Variable: `Loan_Status`
* Classes:

  * `1` = Loan Approved
  * `0` = Loan Rejected

## Project Workflow

* Data Cleaning
* Missing Value Treatment
* Categorical Data Encoding
* Feature Selection
* Train-Test Split
* Handling Imbalanced Data using SMOTE
* Logistic Regression
* Random Forest Classifier
* Model Evaluation

## Libraries Used

* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Imbalanced-learn (SMOTE)

## Models Used

### Logistic Regression

* Accuracy: **78.86%**

### Random Forest Classifier

* Accuracy: **77.24%**

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

## Key Learning

* Data preprocessing is an essential step before training any Machine Learning model.
* Missing values and categorical features must be handled properly.
* SMOTE helps balance imbalanced datasets by generating synthetic samples for the minority class.
* Accuracy alone is not enough to evaluate classification models; Precision, Recall, and F1-Score provide better insights.
